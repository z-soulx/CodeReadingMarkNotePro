Write down that you understood in the code reading work in a place that has a high affinity with code!

Features
  • support for IntelliJ IDEA higher versions (pro added) support for tag grouping, topic notes and sub-topic notes, easy to refine the code query Support customized naming of grouped lists (added by pro) Support codeRemark code notes display (New in pro)
  • Organize your notes by Topic
  • Combine the code line you want to keep to Topic. This makes you always open the code line from note. (like Bookmark feature)
  • Arrange code lines to meaningful order by drag and drop. For example, priority or process flows.

This plugin may not be suitable for projects that change frequently because a point that links with note is static(ex. file path and line number) .
