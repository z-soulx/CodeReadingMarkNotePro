Write down that you understood in the code reading work in a place that has a high affinity with code!

Features

    Tips

  • NEW in v3.7.1: AI Workspace merge pull, .ai/ skeleton creator with preset templates, Quick Reference guide（v3.7.1 新功能：AI 工作空间合并拉取、.ai/ 骨架创建器（含预设模板）、快速参考指南）
  • NEW in v3.6.9: Gutter note popup, Reverse Locate, Trash Bin, distinctive inline annotation color（v3.6.9 新功能：Gutter 笔记弹窗、反定位、废纸篓、行尾注释青色高亮）
  • NEW in v3.6.5: Auto-Sync with Intelligent Conflict Detection - Never lose your notes!（v3.6.5 新功能：智能冲突检测的自动同步 - 永不丢失笔记！）
  • NEW in v3.6.0: Multi-language Support & Enhanced UI（v3.6.0 新功能：多语言支持 & 增强 UI）
  • NEW in v3.5.0: Third-Party Sync with GitHub（v3.5.0 新功能：GitHub 第三方同步）
  • Intelligent search with Pinyin and fuzzy matching - works for both Topics and Bookmarks（智能搜索功能，支持拼音和模糊匹配 - 同时适用于 Topics 和 Bookmarks）
  • Keyboard Shortcuts（快捷键）:
    • Alt+M — Add note to current line（添加笔记到当前行）
    • Alt+G — Open note popup for editing/viewing（打开笔记弹窗编辑/查看）
    • Alt+Delete — Delete note at current line to trash（删除当前行笔记到废纸篓）
    Custom shortcuts: Settings → Keymap → search "Code Reading"（自定义快捷键：设置 → 按键映射 → 搜索 "Code Reading"）
  • Multi-level grouping: Topic → Group → TopicLine with fold/collapse support（多级分组：Topic → Group → TopicLine，支持折叠收缩）
  • Code remarks with inline annotations, gutter icons, and interactive edit popups（代码备注：行尾注释、行首图标、交互编辑弹窗）
  • Native bookmark integration with position sync and batch repair（原生书签联动，支持位置同步和批量修复）
  • GitHub auto-sync with conflict detection for code reading notes（GitHub 自动同步代码阅读笔记，智能冲突检测）
  • Drag-and-drop sorting for Topics, Groups, and TopicLines（拖拽排序 Topic、Group 和 TopicLine）
  • Recoverable Trash Bin for deleted notes（已删除笔记可恢复的废纸篓）

This plugin may not be suitable for projects that change frequently because a point that links with note is static(ex. file path and line number) .
