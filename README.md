- 为什么建立该项目？
  - 原开源项目，长期没有维护，没有新功能且高版本不支持。 
  - 期望： 共同维护，欢迎pr、issues 、持续维护
- 功能列表
  - 支持IntelliJ IDEA 更高版本 (pro新增)
  - 支持标签分组、主题笔记和子题笔记，方便细化代码查询
  - 支持分组后列表的自定义命名(pro新增)
  - 支持codeRemark代码备注展示(pro新增)
